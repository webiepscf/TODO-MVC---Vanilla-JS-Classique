CONSIGNES:

AJOUT D'UNE TÂCHE
  - Ajout d'une tâche quand on clique sur Enter
  - Effet SlideDown à l'ajout
  - Le nombre de tâches à terminer doit se mettre à jour

CHECK et DECHECK D'UNE TÂCHE
  - Quand on check l'input checkbox d'une tâche,
    on l'affiche comme terminée (class completed)
  - Quand on l'affiche en non-terminée si on décheck la checkbox
  - On met à jour le nombre de tâches terminées

MODIFICATION D'UNE TÂCHE
  - Quand on double-clique sur une tâche non terminée, on place
    son contenu dans un input
  - Quand on appuie sur Enter, on vire l'input

SUPPRESSION D'UNE TÂCHE
  - Quand on clique sur la croix à la droite d'une tâche,
    On supprime la tâche avec un effet slideup
  - On met à jour le nombre de tâches non terminées.

LES FILTRES
  - Quand on clique sur un des filtres, on n'affiche que les
    tâches correspondantes.

CLEAR COMPLETED
  - Quand on clique sur ce lien, on vire toutes les tâches terminées
    avec un effet slideup

BONUS:
  - Ne permettre la modification que d'une seule tâche à la fois
